# 🏧 ATM Machine Practice App

Yeh ek simple JavaScript-based ATM machine project hai jo maine apni coding practice ke liye banaya hai.

## 🚀 Features (Isme kya kya hai):
- **Check Balance:** User apna account balance dekh sakta hai.
- **Withdraw Money:** Paise nikalne ka option.
- **Deposit Money:** Naye paise account mein jama karne ka option.

## 🛠️ Tech Stack (Kya use kiya hai):
- HTML5 & CSS3
- JavaScript (Logic & Operations)
